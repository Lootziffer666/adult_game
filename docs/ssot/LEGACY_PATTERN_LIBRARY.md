# LEGACY_PATTERN_LIBRARY

This file preserves useful patterns **without promoting them to canon**.

## Reusable legacy patterns
- Public-friction -> private-aftermath event cadence.
- A/B contrast framing (trust/containment vs attraction/risk) as interaction tension model.
- UI/material mood guidance (minimal, atmospheric, diegetic-feeling).
- Subtle animation doctrine (ambient loops over overt effects).
- Production sequencing templates (foundation -> loop -> systems -> polish).

## Keep as reference only
- Specific explicit scene scripts and exact dialogue lines.
- Tight 14-day promises and date-bound delivery claims.
- Brand/logo exploration documents.
- Any file that presents conflicting product identity as settled fact.
