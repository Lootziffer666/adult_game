# SYSTEMS_OVERVIEW

## Current systems in scope
1. **Temporal system**: minute advancement, phase transitions, day rollover, nightly adjustments.
2. **World reaction system**: rumor/public standing updates and downstream event-context effects.
3. **Relationship system**: hidden A/B/R vectors; no route menu exposure.
4. **Event orchestration**: condition-based + priority-based event triggering, with pending support.
5. **Consequence timing**: immediate and delayed outcomes.
6. **Behavior consistency memory**: style continuity as hidden influence.

## Explicitly out of scope for SSOT baseline
- Canonized story bible.
- Locked final character lore.
- Expanded asset pipeline mandates as gameplay requirements.
