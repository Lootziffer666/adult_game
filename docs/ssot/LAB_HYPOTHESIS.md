# LAB_HYPOTHESIS

## Core hypothesis
A player-facing loop built on interpretation and context can function as a game:

`hint/observation -> interpretation -> action -> world reaction -> new information`

## What is being tested
1. Does the player read reactions as meaningful consequences?
2. Do timing/context shifts change outcomes in understandable ways?
3. Do repeated attempts feel like learning, not brute-force busywork?

## Evidence the hypothesis works
- Different contexts produce legible differences in reaction output.
- Players can form and revise hypotheses from available signals.
- Lightweight hints/memory output improves comprehension without over-guiding.
- Iteration feels purposeful rather than random trial spam.

## What does NOT need to be solved yet
- Final story canon.
- Final setting canon.
- Final character canon.
- Final macro-route architecture.
- Full content scale/polish.
